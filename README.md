## Install dependencies:
pip install -r requirements.txt

## Download checkpoints:
Due to GitHub file size limitations, trained model checkpoints are hosted externally.
Download link: https://drive.google.com/drive/folders/119E9RgpMPcSxCxxfaUGgpo8eIFWi5hn5?usp=share_link

After downloading, place the checkpoint files under:
checkpoints/

## Usage
The entire 3D reconstruction pipeline can be executed using the provided shell script:
bash run.bash
